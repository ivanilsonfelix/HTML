Uso do negrito

Posso fazer o negrito usando a tag B, mas ela nao e semântica

Uso de italico/enfase

posso usar a tag i para fazer um italico, mas não tera valor semantico

é melhor usar a tag em, pois ela é semantica e passa o significado de enfase naquele trecho de texto
